# Proyecto-Discretas
Proyecto de Matemáticas Discretas de Equivalencia entre Álgebra Relacional y SQL

Para la visualizacion del proyecto, se debe ejecutar en el siguiente orden:
    1) Ver que no haya ninguna base de datos creada, si es asi, eliminar el archivo database.db
    2) Cargar la base de datos, en la terminal (con Python y Flask instalados) ejecutar el comando:
                                $ pyhton init_db.py
    3) Despues de salir el mensaje de que se haya cargado correctamente la base de datos, ejecutar el comando:
                                $ python app.py
       Ir al navegador de preferencia y escribir 127.0.0.1:5000 para visualizar la app web
